# Client Onboarding

1. Process starts in `apps/studio/app/tenants/create`.
2. We map an internal GitHub Issue via Agent action.
3. The script automatically provisions the database seed and spins up a Vercel preview deployment.
4. Total time under 8 minutes. No manual Vercel configuration required.
