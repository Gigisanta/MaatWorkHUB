# Natatorio Template Rules

1. The NMS template includes: `clients`, `groups`, `subscriptions`, `attendances`.
2. Emphasize mobile-first design for instructors taking attendance at the pool.
3. Use the shared shadcn/ui components from `@maatwork/ui`.
4. Keep the UI ultra-fast with React Server Components where possible.
