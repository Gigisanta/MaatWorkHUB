# Studio Command Palette (Cmd+K)

1. The central hub for Founders relies heavily on keyboard navigation.
2. When adding a new feature to Studio, always register a corresponding action in `<CmdKPalette />`.
3. Typical actions: "Create new tenant", "Go to billing", "Search client", "Toggle dark mode".
